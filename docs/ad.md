# Reddit post — Maiden

## Title

Maiden — a free, open-source NIKKE Discord bot for looking up characters and converting event times

## Post body

Hey Commanders 👋

I'm in a cluster of NIKKE unions and I wanted to build a bot for our server that pulled together the stuff we look up constantly. It's called **Maiden**, it's **free**, and I wanted to open it up to the community as an open-source tool for any other union or NIKKE community Discord.

**What it does:**

🔎 **`/nikke <name>`** — look up any NIKKE and get it all in one card: profile (weapon, burst + cooldown, class, manufacturer, element), **Prydwen** Story/Bossing/PvP tiers (including the treasure version's ratings), **Nikke Synergy** arena pick & win rates, and build + pull-priority notes — with links back to every source. Nicknames work too, so `rrh` finds Rapi: Red Hood.

🕒 **`/time`** — turn "July 10, 8pm UTC+9" into a Discord timestamp that shows in _everyone's own_ local timezone. No more "wait, what time is that for me?" in the comments.

📰 **Automatic event-time stamps** — point it at your news/tweet channel and it auto-replies to game announcements with the event time in local time (it even handles the official account's `17:00～19:30（UTC+9）` formatting).

💬 **Quotes** — react to a funny message with a set emoji and it saves it; pull up anyone's greatest hits with `/quotes @user`.

Plus quick links to the community guides, and it's all set up by admins with a few `/config` commands.

**Add it here:** https://discord.com/discovery/applications/1523719703950790946
**Source code (open-source):** https://github.com/Infernal-Crack-LED/bakery-bot
**Questions / bugs / suggestions:** support server → https://discord.gg/3Yx4pHB88R

Full disclosure: I'm the dev, it's open-source (so you can see exactly what it does or self-host it), and there are no ads or paywalls — I just wanted it for my own union. Happy to answer anything or take feature requests.

_TL;DR: free NIKKE Discord bot — character lookups, tier/arena data, build notes, and automatic local-time conversion for event times._
